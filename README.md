## 二、洛杉矶三条核心线路，到底差在哪

先把术语翻译成大白话，这一步搞明白了，后面看套餐配置就不会再迷惑。

**CMIN2 线路**：中国移动的 CMI 高端国际骨干网（AS58807），主要优化移动用户的回程。移动用户走这条线到洛杉矶，延迟通常能压到 140ms 上下，晚高峰也基本不掉速。缺点是只对移动友好，电信、联通走过来就普通国际线路水平。

**CN2 GIA 线路**：中国电信的全球互联网接入专线（Global Internet Access），属于电信高端产品线，回程强制走 CN2 骨干，国内电信用户访问延迟稳定、丢包率低。这是国内站长圈子里被吹得最多的一条线，也是 VMISS 洛杉矶老牌系列之一。

**三网优化 TRI**：把上面两条加联通 9929（AS9929，联通的高端骨干）整合到一起，电信走 CN2 GIA、联通走 CUII/9929、移动走 CMIN2，三家运营商各有各的专线，回程都被照顾到。代价是流量配额通常比单线产品小，价格也略高。

> 一句话总结：只用移动选 CMIN2、只用电信选 CN2 GIA、三网用户混着来的选 TRI，但要做好流量配额偏小的心理准备。
